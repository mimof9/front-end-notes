# 前端读书笔记

## JavaScript语言精粹
- 薄而有深度
- 有很多实践被ES5标准采用
- 会涉及到一些标准方法的实现思路，比如数组的reduce方法
- 关于闭包的运用，借助闭包实现模块化的思路等都很棒
- 作者整理的标准类型上的小巧方法集可以快速记忆
